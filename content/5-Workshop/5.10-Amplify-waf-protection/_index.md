#### Goal

In this section, you will enable AWS WAF for the frontend application hosted on AWS Amplify to improve security, reduce exposure to web attacks, and control traffic more effectively.

#### Overview

AWS WAF helps protect your application from common web vulnerabilities, malicious bots, and scanning traffic. After enabling WAF, you can monitor and manage incoming requests through custom rules.

#### Section structure

- Step 1: Access the firewall configuration in AWS Amplify
- Step 2: Configure the firewall options
- Step 3: Review costs and activate the firewall
- Step 4: Wait for the system to associate and finish
