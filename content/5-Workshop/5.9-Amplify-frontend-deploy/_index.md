#### Goal

In this section, you will deploy the GameHub frontend to AWS Amplify by connecting a GitHub repository, configuring the Flutter Web build, and setting up the required environment variables.

#### Overview

AWS Amplify is a suitable service for hosting static or SPA applications such as Flutter Web. After completing the steps below, the frontend will be built and deployed automatically whenever changes are pushed to the selected Git branch.

#### Section structure

- Step 1: Initialize the app on AWS Amplify
- Step 2: Connect the repository and branch
- Step 3: Configure app settings and the amplify.yml file
- Step 4: Configure advanced settings and environment variables
- Step 5: Review and deploy
